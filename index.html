Multiple Choice Questions (MCQs)

Unit 4: Midpoint Ellipse Algorithm

Q1. Which ellipse drawing algorithm is described in the lecture as the best for computer graphics applications?
A
Bresenham Ellipse Algorithm
B
Scan-Line Ellipse Algorithm
C
Midpoint Ellipse Algorithm
D
Polar Coordinate Ellipse Algorithm
Correct Answer: C
The Midpoint Ellipse Algorithm is considered the best for computer graphics applications because it's efficient and uses integer arithmetic.
Q2. According to the lecture, the Midpoint Ellipse Algorithm is considered best primarily because it is:
A
Simple but computationally expensive
B
Accurate, efficient, and requires only a few integer additions per pixel
C
Based on floating-point arithmetic
D
Suitable only for small ellipses
Correct Answer: B
The Midpoint Ellipse Algorithm is accurate, efficient, and requires only integer additions per pixel, making it ideal for rasterization.
Q3. What inputs are required before starting the Midpoint Ellipse Algorithm?
A
Only the center of the ellipse
B
Radius along x-axis and y-axis and the center of the ellipse
C
Diameter along x-axis and y-axis
D
Only the major axis length
Correct Answer: B
The algorithm requires the radius along x-axis (rx), radius along y-axis (ry), and the center coordinates (xc, yc).
Q4. In the algorithm, the ellipse is initially assumed to be centered at:
A
(rx, ry)
B
(1, 1)
C
(0, 0)
D
(xc, yc)
Correct Answer: C
The algorithm initially assumes the ellipse is centered at the origin (0, 0) for simplicity, then translates points to the actual center later.
Q5. What is the first point plotted for the ellipse when centered at the origin?
A
(rx, 0)
B
(0, ry)
C
(ry, 0)
D
(rx, ry)
Correct Answer: B
The algorithm starts at point (0, ry) on the positive y-axis and plots points in the first quadrant.
Q6. The initial decision parameter for Region 1 is denoted as:
A
p20
B
p2k
C
p10
D
pk
Correct Answer: C
The initial decision parameter for Region 1 is denoted as p10 (p1₀).
Q7. Which of the following correctly represents the initial decision parameter for Region 1?
A
p10 = ry² - rx²ry + 0.5rx²
B
p10 = ry² - rx²ry + 0.25rx²
C
p10 = rx² - ry²rx + 0.25ry²
D
p10 = ry² + rx²ry - 0.25rx²
Correct Answer: B
The correct formula for the initial decision parameter in Region 1 is p10 = ry² - rx²ry + 0.25rx².

More Questions

Q8. In Region 1, if the decision parameter p1k < 0, the next point chosen is:
A
(xk, yk − 1)
B
(xk + 1, yk − 1)
C
(xk + 1, yk)
D
(xk − 1, yk)
Correct Answer: C
If p1k < 0, the next point is (xk + 1, yk) - we increment x but keep y the same.
Q9. In Region 1, if p1k ≥ 0, the next point plotted is:
A
(xk, yk − 1)
B
(xk + 1, yk − 1)
C
(xk − 1, yk)
D
(xk + 1, yk)
Correct Answer: B
If p1k ≥ 0, the next point is (xk + 1, yk - 1) - we increment x and decrement y.
Q10. The transition from Region 1 to Region 2 occurs when:
A
x = y
B
x ≥ y
C
2ry²x ≥ 2rx²y
D
2ry²x < 2rx²y
Correct Answer: C
The transition occurs when the slope changes from > -1 to ≤ -1, which is when 2ry²x ≥ 2rx²y.
Q11. The initial decision parameter for Region 2 is calculated using:
A
The first point of the ellipse
B
The center of the ellipse
C
The last point obtained in Region 1
D
A fixed constant value
Correct Answer: C
The initial decision parameter for Region 2 (p20) is calculated using the last point (x, y) obtained in Region 1.
Q12. The decision parameter used in Region 2 is denoted as:
A
p10
B
p1k
C
p20
D
pk
Correct Answer: C
The decision parameter for Region 2 is denoted as p20 (p2₀) initially, and p2k for subsequent points.
Q13. In Region 2, if the decision parameter p2k > 0, the next point selected is:
A
(xk + 1, yk)
B
(xk, yk - 1)
C
(xk + 1, yk - 1)
D
(xk - 1, yk)
Correct Answer: B
If p2k > 0, the next point is (xk, yk - 1) - we decrement y but keep x the same.
Q14. In Region 2, if p2k ≤ 0, the next point chosen is:
A
(xk - 1, yk)
B
(xk, yk - 1)
C
(xk + 1, yk - 1)
D
(xk + 1, yk)
Correct Answer: C
If p2k ≤ 0, the next point is (xk + 1, yk - 1) - we increment x and decrement y.

True / False Questions

Q17. The Midpoint Ellipse Algorithm relies mainly on integer additions per pixel.
True
False
Correct Answer: True
The algorithm uses integer arithmetic with additions and comparisons, avoiding floating-point calculations for efficiency.
Q18. The algorithm begins by assuming the ellipse is centered at the actual screen coordinates.
True
False
Correct Answer: False
The algorithm initially assumes the ellipse is centered at the origin (0, 0) for simplicity, then translates points to the actual center later.
Q19. Region 1 of the ellipse is processed by incrementing x while monitoring the decision parameter.
True
False
Correct Answer: True
In Region 1, x is incremented in each step, and y may be decremented based on the decision parameter.
Q20. In Region 1, y always decreases regardless of the decision parameter value.
True
False
Correct Answer: False
In Region 1, y decreases only when the decision parameter p1k ≥ 0; otherwise, y remains the same.
Q21. The stopping condition for Region 1 is based on comparing scaled x and y values.
True
False
Correct Answer: True
Region 1 continues until 2ry²x ≥ 2rx²y, which is a comparison of scaled x and y values.
Q22. Region 2 processing primarily involves decrementing y values.
True
False
Correct Answer: True
In Region 2, y is decremented in each step, and x may be incremented based on the decision parameter.
Q23. The Midpoint Ellipse Algorithm calculates points independently for all four quadrants.
True
False
Correct Answer: False
The algorithm calculates points only for the first quadrant, then uses symmetry to get points in the other three quadrants.
Q24. Symmetry is used to reduce computational effort in ellipse drawing.
True
False
Correct Answer: True
By exploiting symmetry, we only need to calculate points for one quadrant and mirror them to the other three quadrants.
Q25. The decision parameters p10 and p20 serve the same purpose in both regions.
True
False
Correct Answer: False
p10 and p20 are different decision parameters with different formulas for Region 1 and Region 2 respectively.
Q26. The algorithm finishes by translating the calculated points to the actual ellipse center.
True
False
Correct Answer: True
After calculating points assuming center at (0, 0), we add the actual center coordinates (xc, yc) to each point.
